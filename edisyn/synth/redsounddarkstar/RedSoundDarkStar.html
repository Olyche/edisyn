Red Sound DarkStar and DarkStar XP2

Single Patch Editor

This patch editor works fine on the DarkStar XP2; it likely will not work properly on the DarkStar but will soon once we get a bit more sysex data from DarkStar users. It is not meant to work with the Vocoda.

Communicating with Edisyn Set Protect to OFF (Utility 3). Set the Master MIDI Channel (Utility 8) to the same channel as Edisyn is using.

Gotchas

The DarkStar's sysex is extremely limited.

  • There is no way for Edisyn to request a patch. You will need to send a patch manually from the DarkStar via Save Program (Utility 5). Edisyn also responds to Save Part Voice (Utility 4) and Save All Programs (Utility 6 or 7).

  • Similarly there is no way to request a merge.

  • The DarkStar also has no way to send to a specific patch: you can only send to the current patch memory, or write to a patch.

  • The DarkStar cannot respond to individual patch parameter changes.
