Midi Tuning Standard

IMPORTANT NOTE

MAKE SURE YOU SET THE DEVICE NUMBER APPROPRIATELY IN THE MIDI MENU! It will not work otherwise.

Knob View

Midi Tuning Standard works by assigning a pitch to every midi value from 0-127. Each of these knob pairs represents the frequency to assign to that midi value.

You compute the pitch by choosing an original note from the 12-tone-equal-temperment, A4-440hz, standard modern western tuning as well as an offset in cents from 0-100 above that note. Your resolution in cents is 100/216. The knob panel as it stands is a direct representation of the midi tuning standard sysex and allows you to manually input your desired values. However, this is not the recommended way to set these values. The Tuning menu option has utilities for calculating these knobs in bulk.

Tuning Utilities

EDO
EDO stands for Equal number of Divisions per Octave and is also commonly known as Equal Temperament. Standard western tuning would be 12 EDO with A4 (midi note 69) set to 440hz. Other tunings have their advantages and disadvantages and a decent summary of the topic can be found here. The calculate button gives the frequency of the selected midi number in standard western tuning, but feel free to use any positive number for frequncy for any midi note number as your root, all it is saying is "I want midi number x from my controller to equate to frequency y and to tune the rest of the notes using equal temperment with z divisons per octave."
