Dave Smith Instruments Prophet '08

Single Patch Editor

This editor will work with the Prophet '08 and Prophet '08 Desktop.

Communicating with Edisyn

Set the MIDI channel to the same as being used in Edisyn. Set MParamSend to NRPN. Set M Param Rec to All or NRPN. Set MIDI Program to ON. Set MIDI Sysex to ON. And set MIDIParamLCD to ON.

Hints

  • When you set the destination for a sequencer track to "Osc 1 Freq", "Osc 2 Freq", or "Osc 1&2 Freq", the track dials will change from number values to notes. The lowest note, C0, corresponds to the note being played by the user -- that is, all sequenced notes are equal to or higher than the played note. If you need notes lower than the played note, pitch the oscillator frequencies down.


Gotchas

  • Even if MIDIParamLCD is set to ON, any real-time changes you make won't be reflected on the Prophet '08 LCD unless you first twiddle a knob so the Prophet '08 is in its edit mode. I suggest twiddling the Misc Parameters Select knob.

  • Additionally, even if MIDIParamLCD is on, and the Prophet '08 is displaying parameter edits, you won't see parameter changes from Layer B unless you press "Edit Layer B".

  • If you change Key Sync for some LFO other than #1, the Key Sync light won't change properly, even if that LFO is currently selected on the Prophet '08 front panel. You have to press some other LFO button, then go back to the LFO in question to see the Key Sync button light properly.
