Yamaha FS1R

Voice Patch Editor



Communicating with Edisyn

Set the FS1R's to receive System Exclusive, Bulk Dumps, Bank Selection, and Program Change. That is, set Rcv SysExcl, RcvBulkDump, Rcv BankSel, and Rcv PgmChng to ON. Set the FS1R's Device Number to either ALL or to the Synth ID in Edisyn's MIDI panel. Set Edisyn's send and receive MIDI channels to the FS1R's receive and send channels respectively.

About the FS1R Menu

The menu has several options which hopefully make working with the patch editor more useful:
  • Set Up Test Performance for Part 1 Only   This builds a Performance and sends it to the FS1R where all parts are turned ofF except for Part 1. Part 1 has its Filter Switch turned on so you can test your Voice with the filter, and also has its Receive MIDI Channel set to Edisyn's outgoing MIDI Channel.

  • Work With Part (1,2,3,4)   This causes Edisyn to use the given Part when sending and requesting the current voice patch.You probably want to set this to Part 1.

  • Swap Operators...   This lets you swap two operators, including voiced and/or unvoiced parameters.

  • Copy Operators...   This lets you copy an operator to another, including voiced and/or unvoiced parameters. You can of course copy an operator to another using Edisyn's copy tab menu options, but you can't filter for voiced or unvoiced parameters.

  • Copy Voiced->Unvoiced   This copies relevant voiced parameters to the equivalent unvoiced parameters in an operator.

  • Copy Unvoiced->Voiced   This copies relevant unvoiced parameters to the equivalent voiced parameters in an operator.


How Send to Current Patch and Request Current Patch Work

Voices can't be played alone: they have to be placed in a Part in a performance and played there. To do this, Send to Current Patch sends the voice to the current Performance in the part number specified with the Work with Part... menu above. Similarly, Request Current Patch requests that part.
